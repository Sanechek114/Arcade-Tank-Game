# Arcade Tank Game (Аркадная танковая игра)
Игра про танки сделаная на python.  

## **Цель игры:**  
Пройти все уровни и открыть все предметы.  

## **Описанаие игры**  
Вы упавляете танком и проходите разные локация, уничтожая врагов.  
Механики: Вращение башни, стрельба, езда.  
Вид камеры сверху вниз.  

## **Управление**  
W, S — Вперед, назад  
A, D — Влево, вправо  
ЛКМ — Стрельба  
ESC — Выход, Пауза

## **Зависимости и запуск**
1.  ```pip install arcade``` в терминал, если библеотека не была установлена.
2. Запустите ```main.py```
